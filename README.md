# 🎓 학습 허브 (Learning Hub) - 자기주도 학습 도구

학습 허브는 학생들이 스스로 학습 계획을 세우고, 하루의 배움을 성찰하며 기록할 수 있도록 돕는 웹 기반 자기주도적 학습 플랫폼입니다.

## 🚀 주요 기능

### 1. 학습 할 일 관리 (To-Do List)
- **과목별 색상 지정**: 국어, 수학, 영어, 과학 등 과목별로 고유 색상을 지정하여 학습 계획을 시각적으로 관리할 수 있습니다.
- **할 일 추가/완료/삭제**: 간편하게 과제를 추가하고, 완료 시 체크박스를 통해 성취감을 느낄 수 있습니다.
- **실시간 업데이트**: 추가 및 삭제 시 목록이 즉시 업데이트됩니다.

### 2. 하루 성찰 일기 (Journaling)
- **날짜별 기록**: 달력 기능을 통해 특정 날짜의 학습 내용이나 느낀 점을 기록할 수 있습니다.
- **성찰 뷰어**: 과거에 작성한 일기를 리스트 형태로 한눈에 확인하고 관리할 수 있습니다.
- **덮어쓰기 방지**: 같은 날짜에 기록이 있을 경우 확인 절차를 거쳐 데이터를 보호합니다.

### 3. 데이터 및 디자인
- **로컬 저장소(LocalStorage)**: 작성한 모든 데이터는 브라우저에 안전하게 저장되어, 창을 닫거나 새로고침해도 유지됩니다.
- **프리미엄 UI**: Glassmorphism 디자인과 인터랙티브한 애니메이션을 적용하여 세련된 사용자 경험을 제공합니다.
- **반응형 웹**: PC와 태블릿 등 다양한 화면 크기에 최적화되어 있습니다.

## 🛠️ 기술 스택
- **Frontend**: HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS)
- **Icons**: Font Awesome 6.4.0
- **Fonts**: Google Fonts (Inter, Outfit)

## 📂 파일 구조
- `index.html`: 앱의 구조 및 레이아웃
- `style.css`: 디자인 시스템 및 반응형 스타일
- `app.js`: 할 일 및 일기 관리 로직, 로컬 저장소 연동

## 💻 시작하기
1. 이 저장소의 모든 파일을 다운로드합니다.
2. `index.html` 파일을 더블 클릭하여 웹 브라우저에서 실행합니다.
3. 별도의 서버나 설치 과정 없이 즉시 사용할 수 있습니다.
